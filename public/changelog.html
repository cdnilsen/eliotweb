Changelog

2/22/24:

  • Connected site to www.eliotweb.net.
  • Fixed some bugs that had been bedevilling the word searches, which should now work. Clicking on a verse cite in the English-language search should bring up a box with the text of the KJV even if no Massachusett has been copied yet.
  • Added the text of the KJV's James, 2 Peter and 1 John, which had been overlooked.
  • Added an 'about' page.
  • Added the changelog page.

2/23/24:

  • Added superscriptions to Mayhew's and the KJV Psalms. Superscriptions are given as 'verse zero'.
  • Started populating the word-per-edition tables (no effect on the search engines as of yet).

2/24/24:

  • Fixed a bug in the Massachusett word search where adjacent verses in the list were sometimes grabbing the same text.
  • Implemented a minor cosmetic fix with word-highlight so that highlighting does not also apply to punctuation at the end of the word (so a search for wosketomp will highlight wosketomp, rather than wosketomp,).
